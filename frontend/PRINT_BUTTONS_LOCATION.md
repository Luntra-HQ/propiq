## ğŸ” Troubleshooting

### "I don't see the print button"

**For Deal Calculator:**
- Make sure you're on https://propiq.luntra.one
- Look in the top-right corner of the calculator header
- It's a small printer icon button
- Try clearing your browser cache (Ctrl+Shift+R or Cmd+Shift+R)

**For PropIQ Analysis:**
- Make sure you've completed an analysis (not just opened the modal)
- Scroll to the bottom of the results
- Look for the action buttons section
- You should see "Print Report" and "Export to PDF" side by side

### "Print dialog doesn't open"

1. Check if your browser allows pop-ups from propiq.luntra.one
2. Try a different browser (Chrome, Firefox, Safari, Edge)
3. Check browser console for errors (F12)
4. Ensure JavaScript is enabled

### "Print preview looks wrong"

1. In print dialog, check "More settings"
2. Ensure "Print backgrounds" is unchecked (saves ink)
3. Try both Portrait and Landscape orientation
4. Adjust margins if needed
5. Use "Print to PDF" as alternative
