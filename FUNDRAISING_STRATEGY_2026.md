## Tracking System & Model

### The Fundraising CRM (Notion/Airtable Template)

I recommend creating a **Fundraising Opportunities Tracker** in Notion or Airtable with the following structure:

#### Fields/Columns:

1. **Opportunity Name** (e.g., "Y Combinator X26")
2. **Type** (Accelerator, VC, Angel, Grant, etc.)
3. **Stage Match** (Pre-Seed, Seed, Series A)
4. **Priority** (P1-P5, 1 = highest)
5. **Status** (Not Started, In Progress, Applied, Accepted, Rejected, Watching)
6. **Deadline** (Date field)
7. **Funding Amount** ($50K-500K range)
8. **Equity/Dilution** (% or "Non-dilutive")
9. **Requirements** (MRR, team size, location, etc.)
10. **Application Effort** (Low, Medium, High)
11. **Match Score** (1-5 stars)
12. **Next Action** (Draft application, Record video, etc.)
13. **Contact/Warm Intro** (Name/email if you have a connection)
14. **Notes** (Strategy, pros/cons, etc.)
15. **Resources** (Links to application, guidelines, examples)

#### Views:

1. **Calendar View** - See all deadlines by month
2. **Kanban Board** - Move opportunities through pipeline (Research → Applying → Waiting → Outcome)
3. **Priority View** - Filter by P1/P2 opportunities + upcoming deadlines
4. **Stage View** - Group by revenue stage match
5. **Type View** - Filter by Accelerator, VC, Angel, Grant, etc.

#### Automation/Reminders:

- **4 weeks before deadline:** Reminder to start application
- **2 weeks before deadline:** Reminder to finalize materials
- **1 week before deadline:** Final review reminder
- **After decision:** Update status, send thank-you, request feedback

### Decision Framework: When to Apply

Use this scoring system to prioritize:

**Match Score (1-5 stars):**
- ⭐⭐⭐⭐⭐ Perfect fit (e.g., YC for early-stage SaaS)
- ⭐⭐⭐⭐ Excellent fit
- ⭐⭐⭐ Good fit
- ⭐⭐ Possible but not ideal
- ⭐ Poor fit (skip)

**Priority Calculation:**
```
Priority = (Match Score × Funding Amount × Urgency) / Application Effort

Where:
- Funding Amount: Normalized 1-5 ($50K=1, $500K+=5)
- Urgency: Deadline in <30 days = 3, 30-90 days = 2, 90+ days = 1
- Application Effort: Low=1, Medium=2, High=3
```

**Example:**
