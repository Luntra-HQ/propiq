## üì± The Winning Reel Formula

### Structure (30-60 seconds):

**1. Hook (0-3 seconds)** - Stop the scroll
```
Examples:
- "The 1% rule will lose you money"
- "Stop using spreadsheets for property analysis"

```

**2. Problem (3-10 seconds)** - Relate to pain point
```
- "Most investors waste 3+ hours analyzing each property..."
- "You're using outdated formulas from the 1970s..."
- "Spreadsheets miss critical factors..."
```

**3. Solution (10-45 seconds)** - Provide value
```
- Quick tip or insight
- Show PropIQ in action (screen recording)
- Explain concept visually
```

**4. CTA (45-60 seconds)** - Drive action
```
- "Try PropIQ - link in bio"
- "Free trial at PropIQ dot LUNTRA dot one"
- "Comment DEAL for the link"
```

## üìÖ Posting Strategy (Based on Social Media Examiner Intel)

### Optimal Times:
- **Best:** Tuesday-Thursday, 3:00-5:00 PM ET
- **Good:** Monday/Friday, 12:00-2:00 PM ET
- **Avoid:** Weekends (lower B2B engagement)

### Distribution Sequence (CRITICAL):
1. **Post Reel to feed** (3:00 PM)
2. **WAIT 2-3 hours** (let algorithm push to Explore)
3. **Share to Stories** (5:30-6:00 PM)
4. **Cross-post to YouTube Shorts** (same day or next)
5. **Optional:** Repurpose to TikTok (if audience fits)

**Why this works:**
- Instagram prioritizes one content type at a time
- Immediate Story share kills feed reach
- Waiting allows Explore/Recommendations momentum
