```tsx

// Inside your component where you display PropIQ analysis
const [analysisResult, setAnalysisResult] = useState<PropertyAnalysis | null>(null);

// After analysis completes
useEffect(() => {
  if (dealIQResponse) {
    const analysis: PropertyAnalysis = {
      address: dealIQResponse.address,
      summary: dealIQResponse.summary,

      investment: dealIQResponse.investment,

      <>
        {/* Your analysis display code */}
```

## üêõ Common Issues & Fixes

### Issue: Hero section overlaps existing content

**Fix:** Add `position: relative; z-index: 10` to hero section

```tsx
<section className="relative z-10 bg-gradient-to-br...">
```

### Issue: PDF button doesn't download

**Fix:** Check browser console for errors. Likely causes:
1. `jspdf` not installed: Run `npm install jspdf html2canvas`
2. Analysis data is null: Verify `analysis` prop is passed
3. Browser blocks download: Check browser settings

### Issue: PDF layout is broken

**Fix:** Check analysis data structure. Run:

```tsx
console.log('Analysis data:', analysisData);
```

Ensure it matches the `PropertyAnalysis` interface.
