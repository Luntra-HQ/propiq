## Database Structure (Columns)

### Core Fields
1. **Opportunity Name** (Text)
2. **Type** (Select: Accelerator, VC, Angel, Grant, Cloud Credits, Other)
3. **Stage Match** (Select: Pre-Seed, Seed, Series A, Any)
4. **Priority** (Select: P1, P2, P3, P4, P5)
5. **Status** (Select: Not Started, Researching, Applying, Applied, Accepted, Rejected, On Hold, Watching)
6. **Deadline** (Date)
7. **Funding Amount** (Text: e.g., "$500K", "$100K-250K")
8. **Equity/Terms** (Text: e.g., "7%", "Non-dilutive", "10-15% SAFE")
9. **Requirements** (Text: MRR threshold, location, team size, etc.)
10. **Application Effort** (Select: Low, Medium, High)
11. **Match Score** (Select: ⭐, ⭐⭐, ⭐⭐⭐, ⭐⭐⭐⭐, ⭐⭐⭐⭐⭐)
12. **Next Action** (Text)
13. **Contact/Warm Intro** (Text: name + connection path)
14. **Notes** (Long text)
15. **Resources** (URL)

### Calculated Fields (if using Airtable/Notion formulas)
16. **Days Until Deadline** (Formula: Deadline - Today)
17. **Ready to Apply?** (Checkbox: based on MRR requirements met)
18. **Last Updated** (Auto: Last modified date)

## Views to Create in Your Tracker

### View 1: Priority Dashboard (Default)
- **Filter:** Priority = P1 OR P2
- **Sort:** Deadline (ascending)
- **Show:** All fields
- **Purpose:** Daily view of what matters most

### View 2: Calendar View
- **Group by:** Month (using Deadline field)
- **Show:** Opportunity Name, Priority, Funding Amount, Status
- **Purpose:** See all deadlines by month

### View 3: Kanban Board
- **Group by:** Status
- **Columns:** Not Started | Researching | Applying | Applied | Accepted | Rejected | On Hold
- **Purpose:** Move opportunities through pipeline

### View 4: By Type
- **Group by:** Type
- **Show:** Accelerator, VC, Angel, Grant, etc.
- **Purpose:** Compare options within category

### View 5: Ready to Apply
- **Filter:** Status = "Not Started" AND (Requirements met OR blank)
- **Sort:** Match Score (high to low)
- **Purpose:** See what you can apply to NOW

### View 6: Watching List
- **Filter:** Status = "Watching"
- **Sort:** Match Score (high to low)
- **Purpose:** Track opportunities for when you hit milestones

## Automation Suggestions

### If using Notion:
1. Create a "Deadlines" page with all dates in calendar format
2. Set up "Remind me" for each deadline (4 weeks, 2 weeks, 1 week before)
3. Create a formula property: "Days Until Deadline" = Deadline - Today
4. Create a filter: Show only items with <30 days until deadline

### If using Airtable:
1. Set up automations:
   - When "Deadline" is 4 weeks away → Send Slack/email reminder "Start application for [Opportunity]"
   - When "Deadline" is 2 weeks away → Send reminder "Finalize materials"
   - When "Deadline" is 1 week away → Send reminder "Final review"
   - When "Status" changes to "Applied" → Send confirmation + set follow-up reminder
2. Create filtered views for each reminder window
3. Link to Google Calendar for visual timeline
