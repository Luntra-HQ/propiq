**LinkedIn Post Format:**
- Length: 1,300-3,000 characters
- Hook: First 2 lines (critical)
- Structure: Problem â†’ Insight â†’ Solution â†’ CTA
- Hashtags: 5-7 relevant tags
- CTA: Question or link

**Twitter Thread Format:**
- Length: 7-10 tweets
- Tweet 1: Hook (numbered X/Y)
- Tweets 2-9: Value (one idea per tweet)
- Tweet 10: CTA + link
- Tip: Use line breaks for readability

**Instagram Reel Format:**
- Length: 30-60 seconds
- Hook: 0-3 seconds (stop the scroll)
- Value: 10-45 seconds
- CTA: 45-60 seconds
- Music: Trending audio
